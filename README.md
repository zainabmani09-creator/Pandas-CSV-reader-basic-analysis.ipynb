# Pandas Data Analysis – Project 2

## Description
This project is part of Week 1 – Project 2 of the internship at Syntexchub.
It demonstrates how to use Pandas to read and analyze a CSV dataset.

## Dataset
Titanic machine learning from disaster Dataset (Sample - titanic-machine-learning-from-disaster-metadata.json )

## Concepts Covered
- Reading CSV files using Pandas
- Exploring data using head, tail, info, and describe
- Filtering rows and selecting columns
- Handling missing values
- Creating new columns
- GroupBy, aggregation, and pivot tables
- Merging and concatenating DataFrames
- Advanced operations like cumulative sum, rolling window, rank, and shift

## How to Run
1. Open the notebook `Pandas CSV reader & basic analysis.ipynb`
2. Ensure `Sample - titanic-machine-learning-from-disaster-metadata.json` is in the same folder
3. Run all cells sequentially
